Next-sibling combinator (+)

Селектор '+' вибирає наступного сусіда відносно вказаного елемента.

⁢!DOCTYPE html>
            ⁢html>
            ⁢head>
            
          
           ⁢/head>
            ⁢body>
             ⁢section>
               ⁢h2>⁢/h2>
               ⁢p>⁢/p>
             ⁢/section>
             ⁢section>
               ⁢h2>⁢/h2>
               ⁢p>⁢/p>
             ⁢/section>
            ⁢/body>
            ⁢/html>
            CSS
        h2 + p {
            font-weight: bold;
        }

У цьому прикладі лише ⁢p>, безпосередньо слідуючий за ⁢h2>, буде жирним.

Child combinator (>)

Селектор '>' вибирає елементи, які є прямими дітьми вказаного елемента.

⁢!DOCTYPE html>
        ⁢html>
        ⁢head>
          
        ⁢/head>
        ⁢body>
          ⁢section>
            ⁢h2>⁢/h2>
            ⁢p>⁢/p>
          ⁢/section>
          ⁢section>
            ⁢h2>⁢/h2>
            ⁢div>
              ⁢p>⁢/p>
            ⁢/div>
          ⁢/section>
        ⁢/body>
        ⁢/html>
         
        
        CSS:
        section > p {
            color: red;
        }

У цьому прикладі тільки елементи ⁢p>, які є безпосередніми дітьми ⁢section>, матимуть червоний текст.

Subsequent sibling combinator (~)

Селектор '~' вибирає всі наступні сусідні елементи відносно вказаного елемента.

⁢!DOCTYPE html>
        ⁢html>
        ⁢head>
        ⁢/head>
        ⁢body>
          ⁢section>
            ⁢h2>⁢/h2>
            ⁢p>⁢/p>
            ⁢p>⁢/p>
          ⁢/section>
          ⁢section>
            ⁢h2>⁢/h2>
            ⁢p>⁢/p>
          ⁢/section>
        ⁢/body>
        ⁢/html>  
    
    CSS:
    h2 ~ p {
        font-style: italic;
    }

У цьому прикладі всі ⁢p>, які слідують за ⁢h2>, матимуть курсивний текст.

Descendant combinator (" ")

Селектор-потомок вибирає всі елементи, які є нащадками вказаного елемента.

        ⁢!DOCTYPE html>
        ⁢html>
        ⁢head>
          ⁢style>
            
          ⁢/style>
        ⁢/head>
        ⁢body>
          ⁢section>
            ⁢h2>⁢/h2>
            ⁢p>⁢/p>
          ⁢/section>
          ⁢section>
            ⁢h2>⁢/h2>
            ⁢div>
              ⁢p>⁢/p>
            ⁢/div>
          ⁢/section>
        ⁢/body>
        ⁢/html>
        CSS:
        section p {
        border: 1px solid #000;
      }

У цьому прикладі всі ⁢p>, які є нащадками ⁢section>, матимуть рамку. Ці приклади показують, як ви можете використовувати CSS-селектори разом із семантичними тегами HTML для стилізації різних частин вашої сторінки.
